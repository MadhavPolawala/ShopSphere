# 🛒 ShopSphere – MERN Stack E-Commerce Platform

ShopSphere is a full-stack e-commerce web application built using the MERN stack. It provides a seamless shopping experience with secure authentication, product management, cart functionality, and order processing.

## ✨ Features

### 👤 User Features
- User registration and login (JWT-based authentication)
- Browse products with dynamic listings
- Add/remove items from cart
- Place orders
- Responsive UI for all devices

### 🛠️ Admin Features
- Admin dashboard access
- Add, update, and delete products
- Manage product inventory
- View and manage orders

---

## 🧰 Tech Stack

**Frontend:**
- React.js (Vite)
- Tailwind CSS
- Axios

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (MongoDB Atlas)

**Authentication:**
- JSON Web Tokens (JWT)
